# Eventos Cerca – Prototipo

## Cómo abrir
1. Abre `index.html` en tu navegador (Chrome/Edge/Firefox).
2. Navega con los tabs: Eventos / Buscar / Mi lista / Perfil.
3. Prueba:
   - Aplicar filtros (Buscar).
   - Ver detalle y "Añadir a mi lista".
   - Exportar la lista (Mi lista).

## Accesibilidad (checks sugeridos)
- Navegación con teclado (Tab / Shift+Tab) y foco visible.
- Etiquetas de formularios y landmarks.
- Contraste: usa el modo "alto contraste" en Perfil.
- Herramientas: WAVE y axe DevTools.

> Nota: Es un prototipo educativo sin backend. Los datos se guardan en LocalStorage.
